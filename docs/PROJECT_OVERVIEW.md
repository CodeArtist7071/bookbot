# Project Overview: BookBot

BookBot is a WhatsApp-first SaaS platform designed to help local businesses (salons, clinics, spas, gyms) automate appointment bookings and collect customer reviews through a simple conversational experience.

## Vision
To become the lightweight business automation platform that helps small businesses improve customer experience, increase repeat bookings, and grow their online presence without complex software or manual effort.

## Core Value Proposition
- **WhatsApp-First**: No apps to download; uses the platform customers already use.
- **Review Automation**: Turn every appointment into a potential Google Review.
- **Simplicity**: QR-code-to-booking flow reduces friction.
- **Management**: Centralized dashboard for business owners to track everything.

## Tech Stack
- **Frontend**: React (hosted on Vercel)
- **Backend**: Node.js & Express (hosted on Railway)
- **Database/Auth**: Supabase (PostgreSQL with RLS)
- **Messaging**: Meta WhatsApp Cloud API
- **Payments**: Razorpay
