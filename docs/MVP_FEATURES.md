# MVP Features

The Minimum Viable Product (MVP) focuses on four foundational pillars to provide immediate value to local businesses.

## 1. WhatsApp Appointment Booking
*   **Flow**: Service Selection → Date Picking → Time Slot Selection → Confirmation.
*   **Conflict Management**: Real-time checking against the database to prevent double bookings.
*   **User Experience**: Uses WhatsApp Interactive Messages (Buttons and Lists) for a friction-less UI.

## 2. Automated Review Collection
*   **Trigger**: Sent automatically X hours after an appointment is completed.
*   **Feedback Loop**:
    *   **High Ratings (4-5 stars)**: Prompted to post on Google Maps.
    *   **Low Ratings (1-3 stars)**: Private feedback captured for the owner to address internally.
*   **Reputation Management**: Helps businesses improve their online standing without manual effort.

## 3. Centralized Dashboard (SaaS Portal)
*   **Appointment Calendar**: View and manage all bookings in a list or calendar view.
*   **Customer Directory**: Track customer history, frequency, and feedback.
*   **Service Management**: Add/Edit services, pricing, and duration.
*   **Subscription Management**: Integrated with Razorpay for monthly/yearly SaaS fees.

## 4. QR Code Generation
*   **Function**: Generates a unique WhatsApp "wa.me" link with a pre-filled message.
*   **Use Case**: Placed on shop windows, reception desks, or marketing flyers.
*   **Tracking**: Ability to track which QR codes are driving the most traffic.
