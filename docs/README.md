## Getting Started

To contribute to this project or set up your local development environment, please ensure you have access to:
1.  **Meta Developer Account** (for WhatsApp Cloud API)
2.  **Supabase Project** (for DB and Auth)
3.  **Razorpay Dashboard** (for Payments)
4.  **Railway Account** (for Backend hosting)
